# <b>Infinite Manipulation</b>
___

<p>
<b>Descripción de la Actividad </b>: Cada estudiante realizará una 
serie de ejercicios prácticos de programación que implican la creación, 
manipulación y análisis de listas en un lenguaje de programación 
específico (como Python, Java, C++, etc.). La actividad se 
centrará en tareas como la inserción, eliminación, 
búsqueda, y ordenamiento de elementos en listas.
</p>

### Estructura del proyecto
	- Infinite Manipulation
	  - Program.cs
	  - Models
	    - Vault.cs
	- Readme.md
	- gitignore
	- gitattributes

### Requisitos técnicos para correr el proyecto

+ Tener instalado el SDK de .net 8 : [Descargar](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)

### Como ejecutar el proyecto

<b> Pasos a seguir para ejecutar el proyecto </b>: 

1. Abrir la carpeta contenedora del proyecto desde la terminal.
2. Ir al archivo del proyecto:
~~~
 cd '.\Infinite Manipulation\Infinite Manipulation\'
~~~
3. Ejecutar el proyecto:
~~~
dotnet build
dotnet run
~~~
